# Anti-AI Vocabulary Reference

Words statistically overused by AI. Max 1 per paragraph. If you see a cluster, rewrite.

## Banned verbs
- delve (into) -> explore, look at
- leverage -> use
- optimize -> improve
- utilize -> use
- facilitate -> help
- foster -> encourage, build
- bolster -> strengthen
- underscore -> stress, highlight sparingly
- unveil -> show, introduce
- navigate -> manage, handle
- streamline -> simplify
- enhance -> improve
- endeavour -> try
- ascertain -> find out
- elucidate -> explain

## Banned adjectives
- robust -> strong, solid
- comprehensive -> complete, full
- pivotal -> key, important
- crucial -> important
- vital -> essential
- transformative -> significant
- cutting-edge -> new, modern
- groundbreaking -> new, original
- innovative -> new, creative
- seamless -> smooth
- intricate -> complex
- nuanced -> subtle
- multifaceted -> varied
- holistic -> complete

## Banned transitions
- furthermore -> also
- moreover -> also, and
- notwithstanding -> despite
- that being said -> however, but
- at its core -> essentially
- it is worth noting that -> (just state it)
- in the realm of -> in
- in the landscape of -> in
- in today's [anything] -> currently, now

## Banned openings
- "In today's fast-paced world..."
- "In today's digital age..."
- "In an era of..."
- "In the ever-evolving landscape of..."
- "Let's delve into..."
- "Imagine a world where..."

## Banned conclusions
- "In conclusion..."
- "To sum up..."
- "All things considered..."
- "At the end of the day..."

## Banned structural patterns
- "Whether you're a [X], [Y], or [Z]..."
- "It's not just [X], it's also [Y]..."
- Starting sentences with "By" + gerund: "By understanding X, you can Y..."

## Em dash rules
- Max 1-2 per entire page
- Use commas for parenthetical info instead
- Use colons to introduce explanations
- Use parentheses for supplementary info

## Self-check process
1. Read text aloud — if it sounds unnatural, rewrite
2. Count AI vocab words per paragraph — max 1
3. Check for three-adjective clusters — pick one
4. Verify sentence case headings
5. Confirm final paragraph is a next step, not a summary
